## Commandes github
Ouvrir un terminal, aller dans le bon dossier, exemple :
```
cd Documents/git_1A/PROJET_ELECTRONIQUE_MIDI/
```

```

```

Pour pousser des modifications locales
```
git add .
git commit -m "Message à remplacer"
git push
```
